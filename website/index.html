Features

강력한 기능

문서 변환에 필요한 모든 기능을 하나의 앱에서 제공합니다

완벽한 문서 변환

HWPX, DOCX, XLSX, PPTX, PDF 등 모든 문서를 HTML로 변환합니다. 원본 서식과 레이아웃을 최대한 보존합니다.

이미지 PDF OCR

스캔본 문서도 Upstage AI로 완벽한 HTML 변환이 가능합니다. 최첨단 OCR 기술로 텍스트를 정확히 인식합니다.

Gemini 3.0 Flash 자동 교정

AI가 원본과 대조하여 오류를 자동 교정합니다. 누락된 텍스트, 잘못된 서식을 지능적으로 수정합니다.

WYSIWYG 에디터

변환된 HTML을 직접 편집할 수 있습니다. 표 편집 및 서식 지원으로 완벽한 결과물을 만들어 보세요.

마크다운 변환

HTML을 마크다운으로 자동 변환하여 저장합니다. 다양한 플랫폼에서 활용할 수 있는 깔끔한 마크다운을 생성합니다.

원클릭 설치

설치파일 하나로 모든 설정이 자동 완료됩니다. 복잡한 설정 없이 바로 사용할 수 있습니다.

How It Works

간단한 3단계

복잡한 과정 없이 3단계만으로 문서 변환이 완료됩니다

1

파일 선택

변환할 문서 파일을 선택하거나 드래그 앤 드롭으로 추가합니다

2

자동 변환

AI가 문서를 분석하고 OCR과 자동 교정을 수행하여 HTML로 변환합니다

3

편집 / 저장

WYSIWYG 에디터로 결과를 편집하고 HTML 또는 마크다운으로 저장합니다
